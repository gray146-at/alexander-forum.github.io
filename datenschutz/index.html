Hosting und technische Bereitstellung

Diese Website wird als statische Website über GitHub Pages bereitgestellt. Beim Aufruf der Website können technisch notwendige Verbindungsdaten verarbeitet werden, etwa IP-Adresse, Zeitpunkt des Aufrufs, angefragte Datei, Browser- und Betriebssysteminformationen. Diese Verarbeitung ist erforderlich, um die Website auszuliefern, sicher zu betreiben und technische Fehler zu erkennen.

Kontaktaufnahme

Wenn Sie uns per E-Mail kontaktieren, werden die von Ihnen übermittelten Daten zur Bearbeitung der Anfrage und für mögliche Anschlussfragen verarbeitet. Ohne diese Daten kann die Anfrage nicht beantwortet werden.

Basis-Statistik mit Simple Analytics

Wir nutzen Simple Analytics für eine datenschutzfreundliche Basis-Statistik der Website. Simple Analytics arbeitet nach eigener Darstellung ohne Tracking-Cookies und ohne Erstellung personenbezogener Nutzerprofile. Erfasst werden aggregierte Nutzungsinformationen, um grob zu verstehen, welche Seiten aufgerufen werden und wie die Website genutzt wird.

Optionale erweiterte Analyse mit Microsoft Clarity

Zusätzlich kann nach Ihrer Einwilligung Microsoft Clarity geladen werden. Clarity hilft uns, Nutzungsverhalten besser zu verstehen, etwa über Klicks, Scrollverhalten, Sitzungen und technische Darstellungsprobleme. Dabei können Nutzungsdaten verarbeitet und Analyse-Cookies gesetzt werden. Clarity wird auf dieser Website erst geladen, wenn Sie im Banner der optionalen Statistik zustimmen.

Sie können Ihre Entscheidung jederzeit über den Link „Datenschutz-Einstellungen“ im Footer erneut öffnen und ändern.

Rechtsgrundlagen

Die technische Bereitstellung der Website erfolgt auf Grundlage berechtigter Interessen an einem sicheren und funktionsfähigen Webauftritt. Die optionale erweiterte Analyse mit Microsoft Clarity erfolgt nur auf Grundlage Ihrer Einwilligung.
